**Simple Messenger** — это легковесный, быстрый и защищенный кроссплатформенный мессенджер с клиент-серверной архитектурой. Создан на Python с использованием UI-фреймворка [Flet](https://flet.dev/).

Проект подходит для развертывания в корпоративной локальной сети (LAN) или на личном VPS (WAN) без зависимости от сторонних сервисов.

## Ключевые возможности

* **Безопасность:** Весь трафик между клиентом и сервером защищен SSL/TLS шифрованием. Регистрация закрыта секретным кодом сервера.
* **Мультиаккаунт:** Бесшовная синхронизация. Читайте, пишите и удаляйте сообщения одновременно с разных клиентов и изменения отобразятся везде.
* **Обмен файлами:** Поддержка отправки и скачивания любых файлов и изображений прямо в чате (с ограничением в `50 мб`).
* **Глобальное удаление:** Удаление сообщения стирает его у всех собеседников и физически удаляет прикрепленные файлы с жесткого диска сервера.
* **Нативная интеграция с ОС:**
  * **macOS:** Сворачивание в Dock, системные уведомления AppleScript и поддержка бейджей.
  * **Windows/Linux:** Сворачивание в системный трей, Balloon-уведомления.

## Структура проекта

Проект разделен на две независимые части:

* **[`/server`](./server/)** — Асинхронный TCP-сервер (asyncio + SQLite). Упакован в Docker. Подробная инструкция по развертыванию сервера находится в [**Документации сервера**](./server/README.md).
* **[`/client`](./client/)** — Графическое приложение (Flet). Построено по паттерну MVC. Подробная инструкция по развертыванию клиента находится в [**Документации клиента**](./client/README.md).
