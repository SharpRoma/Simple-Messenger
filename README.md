**Simple Messenger** — это легковесный, быстрый и защищенный кроссплатформенный мессенджер с клиент-серверной архитектурой. Создан на Python с использованием UI-фреймворка [Flet](https://flet.dev/).

Проект подходит для развертывания в корпоративной локальной сети (LAN) или на личном VPS (WAN) без зависимости от сторонних облачных сервисов.

## Ключевые возможности

* **Безопасность и Шифрование:** Весь трафик между клиентом и сервером защищен SSL/TLS шифрованием. Регистрация закрыта секретным кодом сервера. Пароли клиентов хранятся в зашифрованных системных хранилищах (Keyring / Keychain).
* **Мультиаккаунт и Синхронизация:** Бесшовная работа с нескольких устройств.
* **Обмен файлами:** Поддержка отправки и скачивания любых файлов и изображений в чате.
* **Глобальное удаление:** Удаление сообщения стирает его у всех собеседников и физически удаляет прикрепленные файлы с жесткого диска сервера.
* **Нативная интеграция с ОС (Адаптеры):**
  * **macOS:** Сворачивание в Dock, безопасные системные уведомления AppleScript без риска инъекций.
  * **Windows:** Сворачивание в системный трей (pystray) с поддержкой динамических бейджей непрочитанных сообщений и нативных уведомлений.

## 📂 Структура проекта

Проект разделен на две независимые части:

* **[`/server`](./server/)** — Асинхронный TCP-сервер (asyncio + SQLite). Упакован в Docker для развертывания в один клик. Подробная инструкция находится в [**Документации сервера**](./server/README.md).
* **[`/client`](./client/)** — Графическое Desktop-приложение (Flet). Построено по компонентной архитектуре (MVC/MVVM). Подробное описание устройства клиента находится в [**Документации клиента**](./client/README.md).
