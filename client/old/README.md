# Simple Messenger — Клиентская часть

Этот модуль представляет собой кроссплатформенное графическое приложение (Desktop GUI), написанное на Python с использованием фреймворка [Flet](https://flet.dev/). Приложение взаимодействует с сервером по TCP-протоколу, используя асинхронные сокеты и SSL/TLS-шифрование.

## Архитектура (MVC)

Для удобства поддержки и масштабируемости, код клиента строго разделен на функциональные модули. Это предотвращает создание монолитных "god-файлов" и позволяет легко менять дизайн, не затрагивая сетевую логику.

* **`main.py`** (Точка входа)
  * Отвечает за инициализацию приложения.
  * Управляет многопоточностью, чтобы развести графический движок `Flet` и системный трей `pystray` по разным потокам (критически важно для macOS).
  * Обрабатывает системные события ОС (перехват крестика, разворачивание из трея).
* **`gui.py`** (Представление / View)
  * Содержит всю логику отрисовки интерфейса (экраны авторизации, чата, модальные окна настроек).
  * Управляет буфером обмена, загрузкой и сохранением файлов (`FilePicker`).
* **`network.py`** (Модель сети / Network Layer)
  * Изолированный слой сокетов (`asyncio`).
  * Настраивает SSL-контекст, отправляет и принимает JSON-пакеты, декодирует Base64-файлы.
* **`config.py`** (Конфигурация и ОС)
  * Отвечает за определение операционной системы.
  * Управляет сохранением локальных настроек (`settings.json`) в защищенных системных директориях.
  * Прописывает программу в автозапуск (через Реестр Windows или LaunchAgents macOS).

## Нативные функции операционных систем

Приложение максимально адаптируется под ту операционную систему, в которой запущено:

| Функция | Windows / Linux | macOS |
| :--- | :--- | :--- |
| **Уведомления** | Системный центр уведомлений (`plyer` + генерация `.ico`) | Нативные push-уведомления (через AppleScript / `osascript`) |
| **Сворачивание** | Прячется в Системный трей (возле часов) | Элегантно сворачивается в Dock-панель |
| **Индикация** | Красный кружок (badge) на иконке в трее | Красный кружок на иконке в трее (меню-баре) |
| **Автозапуск** | Запись в `HKEY_CURRENT_USER\...\Run` | Создание `.plist` агента |

## Хранение локальных данных

Приложение ничего не сохраняет в папке установки. Все картинки (сгенерированные иконки трея) и файл `settings.json` (где хранится сохраненный логин и параметры автозапуска) автоматически размещаются в системной директории пользователя:
* **Windows:** `%APPDATA%\SimpleMessenger`
* **macOS:** `~/Library/Application Support/SimpleMessenger`

## Запуск из исходного кода

Для запуска клиента в режиме разработчика:

1. Откройте терминал в корне проекта (на уровень выше папки `client`).
2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv .venv
   # Windows: .venv\Scripts\activate
   # macOS/Linux: source .venv/bin/activate
   ```
3. Установите зависимости из файла `requirements.txt`:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите клиент:
   ```bash
   python client/main.py
   ```

## Сборка (Компиляция в .exe / .app)

Для компиляции скриптов в готовое приложение для конечных пользователей, в корне проекта предусмотрены автоматизированные CI-скрипты. 

**Важно:** Перед сборкой убедитесь, что в корне проекта лежит картинка `app_icon.png` (ваша иконка приложения).

### Windows (`build_win.bat`)
Скрипт автоматически:
1. Находит и активирует виртуальное окружение.
2. Вызывает упаковщик `flet pack` с правильными флагами метаданных (чтобы в Панели задач писалось "Simple Messenger", а не "flet").
3. Очищает временные папки `build` и файлы `.spec`.
* **Результат:** Файл `dist/SimpleMessenger.exe`.

### macOS (`build_mac.sh`)
В macOS используется сложная архитектура сборки, чтобы скрыть внутренний движок Flutter и обеспечить нативное поведение. Скрипт:
1. Конвертирует `app_icon.png` в маковский формат `AppIcon.icns` через встроенную утилиту `sips`.
2. Инжектирует флаг `LSUIElement` в `Info.plist`, чтобы скрыть Python-обертку из Dock.
3. Жестко заменяет иконку и название внутреннего движка.
4. Переподписывает приложение сертификатом (codesign), чтобы macOS не блокировала его.
5. Перезагружает кэш Dock.
* **Результат:** Папка-приложение `dist/SimpleMessenger.app`.
